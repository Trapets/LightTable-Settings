# LightTable settings
=====

My user.behaviors and user.keymap files for the [LightTable](http://www.lighttable.com) (since v.0.7.2) editor.

## Install
Copy all files to:
* Windows `"USER"\AppData\Local\LightTable\User\`
* Linux `/home/"USER"/.config/LightTable/User/`
